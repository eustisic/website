Beekeeper

Beekeeper is an open source tool for rapid deployment of a virtual wait room. Beekeeper is a back end as a service (BaaS) designed to handle bursty traffic from one-off events. It provides an easy-to-deploy and ready-to-scale solution clients serving one off events (such as a sale or promotion) with bursty traffic.

For the application developer, Beekeeper abstracts away the complexity of setting up a virtual wait room. When deployed it provides a URL that can be used as a promotional link to direct user traffic through the Beekeeper infrastructure and to a specified endpoint at some pre-set rate.
